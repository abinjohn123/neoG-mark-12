# Fun With Triangles!

An app all that's all about triangles. Created for the Mark12 project of neoG.camp LevelZero.

## Features Included

1. Check if three input angles make a triangle
2. A fun quiz about triangles
3. Calculate hypotenuse
4. Calculate area

## Tech Stack Used

* HTML
* CSS
* JavaScript
